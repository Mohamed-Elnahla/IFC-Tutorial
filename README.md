# IFC Tutorial

A static GitHub Pages site for learning IFC (Industry Foundation Classes), built as a single-page HTML tutorial.

## Live Site

After enabling GitHub Pages in repository settings, this project is served from `index.html`.

## Project Structure

- `index.html`: Main tutorial page and all embedded styles/scripts.

## Local Development

Because this is a static site, you can open `index.html` directly in a browser.

For a better development experience, run a local static server:

```bash
# Python 3
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. Go to `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select your branch (usually `main`) and root folder (`/`).
5. Save and wait for GitHub Pages to publish.
